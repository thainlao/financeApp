# Финансовое приложение для сборов статистика и занесения данных.
---

Для запуска

```
git clone https://github.com/thainlao/financeApp
cd .\frontend\
npm run dev

cd .\backend\
npm run start:dev

```
---

В стеке: 
* Node JS(Nest)
* React
* Typescript
* Redux Toolkit
* MongoDB
* CSS
